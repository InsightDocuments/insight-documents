# InsightDocuments

InsightDocuments is a minimal viable product (MVP) to demonstrate the value of a PDF-based document search and natural language query tool. It allows users to:
- Upload a PDF file.
- Extract its content.
- Ask natural language questions to retrieve specific answers and references.

## Features
- PDF upload and text extraction.
- Natural language query processing using GPT integration.
- Clear reference links to document sources.
